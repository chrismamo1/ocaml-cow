val color_to_string : ?fmt:color_fmt -> color -> string
color_to_string ~fmt:f c converts a value c of type Cow.Css.color to a string using the format f (either `Hex for values such as "#a0b0c0" or `Rgb for values such as "rgb(42,42,42)"). Uses printf internally.
val color_of_string : ?fmt:color_fmt -> string -> color
color_of_string ~fmt:f s converts a string s to a Cow.Css.color based on the format f. Internally uses Scanf.sscanf for parsing.

val polygradient : gradient_type ->
       ?behaviour:t -> ?low:t -> ?high:t -> unit -> t
Emit a CSS gradient style that can be either a linear gradient or a radial gradient (at the moment it will by default produce a centered circular gradient if asked for a radial gradient). For example, a call of polygradient `Radial <:css<circle>> <:css<#ff0000>> <:css<#00ff00>> will produce a red and green centered circular radial gradient.
val gradient : ?low:t -> ?high:t -> unit -> t
Emit a CSS gradient style that linearly interpolates between the low and high colors moving from top to bottom
low : the color to place at the bottom of the gradient. default is #0a0a0a
high : the color to place at the top of the gradient. default is #ffffff
